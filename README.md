# AI Video Workbench (爆款视频工作台)

这是一个完整的短视频投流/爆款带货视频的全流程辅助工作台，支持各个核心环节的 AI 自动化流转。

## 最近更新内容：Step 2 一键全流程

### 1. Step 2 一键全流程说明
现在的 Step 2 页面集成了自动化的流转管道，帮助用户一键完成 从原始文案到播客定制提示词 的转化：
- **顺序执行：** 依次执行 扩写 → 质量检测 → 生成NBLM提示词。
- **单独重跑：** 对于任一执行不理想的步骤，可以单独点击对应的“单独重跑”按钮进行微调。
- **本地持久化：** 每步的内容、评分结果及配置将自动保存在浏览器的本地缓存中，刷新页面不会丢失工作进度。

### 2. 质量门禁规则
我们引入了针对 AI 扩写稿的“质量门禁”检测自动化环节：
- **严格打分：** 扩写稿生成后，由检测模型进行 100 分制打分。
- **阈值控制：** 系统默认设定 75 分 为通过门槛 (可自行下拉修改为 70 或 80)。
- **拦截反馈：** 只有在总分达到或高于指定阈值时，管道才会放行进入提示词输出环节。如果低于阈值，会被拦截并要求依据反馈建议修改后重跑。

### 3. NotebookLM 提示词策略 (主干+赛道尾巴)
对 NotebookLM 常常失控或产生大量废话的痛点进行了系统级抑制：
- **主干统领：** 所有的提示词采取统一的主干核心，要求：不偷换目标、三段推演结构（结论->理由->可执行做法）、控制发散、缩减废话并限定时长（3-5分钟）等硬指标。
- **赛道尾巴策略：** 系统会结合当前选择的具体赛道（如情感、健康、商业）智能化追加不同的“尾巴”约束。
  - **情感类：** 限定探讨边界及情绪压力，避免强行加入原生家庭冲突。
  - **健康类：** 纯探讨生活方式建议，绝对不输出诊断结果，不脑补乱用数据。
  - **商业类：** 深入体验式呈现及场景使用感，不含绝对化功效承诺。
